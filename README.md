Sistema de Controle de Doação de Sangue

- Projeto tem como objetivo facilitar e incentivar a doação de sangue, facilitando o uso com uma interface simples e intuitiva, abrangendo usuários de todas as idades.
- Foram utilizadas as seguintes tecnologias: HTML e CSS.
